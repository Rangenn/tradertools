TraderTools.

Для работы базы необходимо:
 - исполнить скрипт doc/script.sql или сгенерировать скрипт из модели doc/TTdb.dm2 программой Toad Data Modeller 2.24.
 - в каталоге postgresql8.3/bin в командной строке исполнить: createlang plpgsql tradertools, где tradertools - имя базы. Это активирует язык plpgsql, необходимый для работы функций БД.

Приложение состоит из нескольких инструментов (tools), находящихся в /controller:
DataTransmitTool - утилита для импорта данных, работает в тестовом режиме
SupplyListTool - инструмент для работы со складом (товары, остатки и т.д.)
CustomerManagingTool - инструмент для работы с документами клиентов/поставщиков (счета, платежи, заказы) 

Заполнить прайс-лист тестовыми данными можно так:
controller/DataTransmitTool -> импорт прайс ока ип тихонов -> выбираем doc/книга1.xls -> ждем до появления сообщения об окончании импорта.
